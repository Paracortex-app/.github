### What is this?

Paracortex is an open source desktop application for cognitive augmentation. It sits alongside your workflow, not in front of it. Where most AI tools optimize for automation and delegation, Paracortex optimizes for understanding.

Two parallel intelligences. Human and AI. Flowing alongside each other, exchanging ideas, each maintaining identity.

> Alone you can get things done fast, but together you can go further.
